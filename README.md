# Kritigy Portfolio

A personal portfolio website built with React and Vite.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Build

```bash
npm run build
```
